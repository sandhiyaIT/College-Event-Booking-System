# College Event Booking System

## Overview

A console-based College Event Booking System developed using Python and Oracle Database. The system allows students to register, view events, book tickets, view bookings, and cancel bookings. Administrators can manage events and generate reports.

## Technologies Used

* Python
* Oracle Database
* OracleDB Python Driver

## Features

* Student Registration and Login
* Event Management
* Ticket Booking
* Booking Cancellation
* Report Generation
* Database Integration

## Project Structure

* admin.py – Admin operations
* student.py – Student operations
* booking.py – Booking management
* report.py – Reports and analytics
* database.py – Database connection
* main.py – Application entry point

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Configure Oracle Database.

3. Execute SQL scripts from the sql folder.

4. Run:
   python main.py
